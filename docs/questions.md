# Questions Project Memory Should Answer

## Decision Questions

- Why was this decision made?
- Who proposed it?
- What alternatives were discussed?
- Why was this approach rejected?
- Was everyone aligned?

---

## Timeline Questions

- What happened before this decision?
- What happened afterwards?
- Which incident triggered this change?

---

## Knowledge Questions

- Has this happened before?
- Which services are affected?
- Which documents mention this feature?

---

## Future Questions

These questions will guide future development.

Whenever we build a new feature, it should help answer one or more of the questions above.
